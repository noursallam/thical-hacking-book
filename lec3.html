1. Explain Network Scanning Concepts

(شرح مفاهيم مسح الشبكة)

Overview of Network Scanning

(نظرة عامة على مسح الشبكة)

Discover Live Hosts

Identify devices actively connected to the network.

(تحديد الأجهزة المتصلة بالشبكة بشكل نشط)

Discover IP Addresses

Understand the network's address allocation.

(فهم تخصيص عناوين IP في الشبكة)

Identify Open Ports and Services

Reveal services that might be exploitable.

(الكشف عن الخدمات التي قد تكون قابلة للاستغلال)

Find Vulnerabilities

Detect weak configurations or outdated systems.

(الكشف عن التكوينات الضعيفة أو الأنظمة القديمة)

TCP/IP Communication

(اتصال TCP/IP)

TCP Flags

  • URG (Urgent): Data must be processed immediately.
  • PSH (Push): Transfers all buffered data immediately.
  • ACK (Acknowledgment): Confirms the receipt of data packets.
  • SYN (Synchronize): Used to start a connection (handshake).
  • FIN (Finish): Closes the connection.
  • RST (Reset): Resets a connection if an error occurs.

(أعلام TCP: URG, PSH, ACK, SYN, FIN, RST)

Three-Way Handshake

(مصافحة ثلاثية الاتجاهات)

  1. Client Sends SYN: Initiates a connection to a server by sending a SYN packet.
  2. Server Responds with SYN-ACK: Confirms readiness by sending SYN and ACK packets.
  3. Client Sends ACK: Acknowledges the server's response, and the connection is established.

(1. العميل يرسل SYN 2. الخادم يرد بـ SYN-ACK 3. العميل يرسل ACK)

2. Revision of Networking Protocols

(مراجعة بروتوكولات الشبكة)

ARP (Address Resolution Protocol)

(بروتوكول تحليل العناوين)

ARP is used to resolve IP addresses to MAC addresses within a local network.

(يستخدم ARP لتحويل عناوين IP إلى عناوين MAC داخل الشبكة المحلية)

ICMP (Internet Control Message Protocol)

(بروتوكول رسائل التحكم في الإنترنت)

ICMP is used for diagnostic purposes, such as ping and traceroute.

(يستخدم ICMP لأغراض تشخيصية مثل ping و traceroute)

TCP vs UDP

(مقارنة بين TCP و UDP)

  • TCP: Reliable, connection-oriented protocol.
  • UDP: Unreliable, connectionless protocol.

(TCP: موثوق وموجه للاتصال | UDP: غير موثوق وبدون اتصال)

3. Use Various Network Scanning Tools

(استخدام أدوات مسح الشبكة المختلفة)

Nmap

(أداة Nmap)

Nmap is a powerful network scanning tool used for discovering hosts and services on a network.

(Nmap هي أداة قوية لمسح الشبكة تستخدم لاكتشاف المضيفين والخدمات على الشبكة)

Hping3

(أداة Hping3)

Hping3 is a network tool used for packet crafting and network testing.

(Hping3 هي أداة شبكة تستخدم لإنشاء الحزم واختبار الشبكة)

Metasploit

(أداة Metasploit)

Metasploit is a penetration testing framework used for exploiting vulnerabilities.

(Metasploit هي إطار عمل لاختبار الاختراق يستخدم لاستغلال الثغرات)

4. Demonstrate Various Scanning Techniques for Host Discovery

(إظهار تقنيات المسح لاكتشاف المضيفين)

ARP Ping Scan

(مسح ARP Ping)

Uses ARP requests to identify active hosts on a local network.

(يستخدم طلبات ARP لتحديد المضيفين النشطين على الشبكة المحلية)

ICMP Ping Scan

(مسح ICMP Ping)

Sends ICMP Echo requests to determine if a system is active.

(يرسل طلبات ICMP Echo لتحديد ما إذا كان النظام نشطًا)

TCP SYN Scan

(مسح TCP SYN)

Sends SYN packets to check for open ports and active hosts.

(يرسل حزم SYN للتحقق من المنافذ المفتوحة والمضيفين النشطين)
